# Contributing

## Local setup

Python 3.10+ required. No external dependencies for the main module.

```
git clone https://github.com/snapsynapse/prompter-kit.git
cd prompter-kit
pip install pytest
python3 -m pytest tests/ -v
```

## Making changes

- One logical change per PR.
- Add or update tests for any changed behavior.
- Update CHANGELOG.md under `[Unreleased]`.
- Run the test suite before submitting.

## Commit style

Short imperative subject line, e.g. `Add export-all collision fix`. No period.

## Questions

Open a GitHub Discussion or file an issue.
